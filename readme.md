### Funciones
- Subir Archivo

- Eliminar Archivo
- Descargar Archivo


### Como Usar
```bash
# Clonar el repositorio
$ git clone https://github.com/frank-hz/file-uploader-nodejs

# Ir a la carpeta
$ cd file-uploader-nodejs

# Instalar las dependencias
$ npm install

# Iniciar la aplicacion
$ npm run start
```
